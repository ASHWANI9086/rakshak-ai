Rakshak AI – Privacy-Focused On-Device AI Assistant

Built a next-generation AI-powered productivity assistant that runs entirely on the user’s device, ensuring complete data privacy and zero reliance on external APIs. Implemented local execution of LLM, Speech-to-Text, and Text-to-Speech models using WebGPU/WebAssembly, eliminating the need to send sensitive data to cloud servers. Designed a secure, low-latency system capable of real-time interactions directly in the browser, enhancing performance while maintaining full user data ownership.
